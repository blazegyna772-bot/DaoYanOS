前端技术栈：

类别	技术
框架	Next.js 16.1.6 (App Router)
React	19.2.3
语言	TypeScript 5
样式	Tailwind CSS 4
UI 组件	@base-ui/react + shadcn/ui
图标	Lucide React
工具	Turbopack (开发模式), ESLint 9
特点：

使用 Turbopack 替代 Webpack（更快）
@base-ui/react - Radix UI 的新版本，无头组件库
shadcn CLI 工具用于管理组件
Tailwind CSS v4 使用新的 @tailwindcss/postcss 包

后端技术栈：

类别	技术
运行时	Node.js (通过 Next.js)
API 框架	Next.js App Router (API Routes)
存储	本地文件系统 (JSON 文件)
数据格式	JSON
架构	本地优先 (Local-first)
存储结构：


data/projects/
├── {project_id}/
│   ├── meta.json          # 项目元数据、剧本、分镜
│   ├── assets/            # 角色/场景/道具文件
│   └── outputs/           # 生成的图片/视频
API 端点：

GET/POST /api/projects - 列出/创建项目
GET/PATCH/DELETE /api/projects/[id] - 项目 CRUD
POST /api/projects/[id]/assets - 资产管理
POST /api/analyze-script - 剧本分析
特点：

无数据库 - 纯文件系统存储
无外部依赖 - 不依赖 Redis、MongoDB 等
可配置存储路径 - 通过 LOCAL_STORAGE_PATH 环境变量
JSON 序列化 - 数据以 JSON 格式持久化
