# 界面架构与功能要求

1. 左侧：全局导航栏 (Sidebar)

- 包含七个大类：管理、剧本、资产、分镜、AI生成、后期、配置。

2. 中部：多标签工作区 (Workspace)

3. 右侧：上下文折叠工具面板 (Inspector/Tools Panel)

- 视觉要求：工具区的控件要紧凑，字号偏小（text-sm 或 text-xs），体现参数配置的专业感。
